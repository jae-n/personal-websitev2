# Personal Website v2

A personal website built with React + Vite. Includes Tailwind CSS and project configuration files for a modern frontend setup.

## Features
- Vite + React
- Tailwind CSS
- Project structure with src/, app/, and styles/

## Prerequisites
- Node.js 16 or newer
- pnpm (recommended)

## Install
```bash
pnpm install
```

## Development
```bash
pnpm dev
```

## Build
```bash
pnpm build
```

## Preview
```bash
pnpm preview
```

## Project Structure
- index.html — App entry
- src/ — Source files
- vite.config.ts — Vite configuration
- package.json — Scripts and dependencies
